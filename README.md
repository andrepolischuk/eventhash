# Eventhash

  Hashchange event shim

## Instalation

  Browser:

```html
<script src="https://cdn.rawgit.com/andrepolischuk/eventhash/0.1.0/eventhash.min.js"></script>
```

  Component(1):

```sh
$ component install andrepolischuk/eventhash
```

  Npm:

```sh
$ npm install eventhash
```

## API

### eventhash(fn)

  Set onhashchange handler

```js
eventhash(function() {

});
```
