# eventhash

  > Hashchange event shim

## Instalation

```sh
$ npm install --save eventhash
$ component install andrepolischuk/eventhash
```

## API

### eventhash.bind(fn)

  Bind event `hashchange` handler

```js
eventhash.bind(function() {

});
```

### eventhash.unbind(fn)

  Unbind event 'hashchange' handler

```js
eventhash.unbind(fn);
```
